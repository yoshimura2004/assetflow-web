# 📈 AssetFlow - 자산 관리 플랫폼 (Frontend)

개인 자산 및 주식 포트폴리오를 효율적으로 관리하기 위한 웹 애플리케이션입니다.

## 🚀 주요 기능
* 실시간 주식 데이터 시각화 (TradingView API 활용)
* 다국어 지원 (한국어, 일본어, 영어) - `i18next` 활용
* 자산 추가/수정/삭제 및 실현 손익 자동 계산

## 🛠 Tech Stack
* **Frontend:** React, Tailwind CSS, i18next
* **Communication:** REST API (Spring Boot Backend 연동)
